# Подсказки по коммандной строке

Комманда смены директории
```sh
cd c:\folder_name
```

Комманда отображения текущей директории. MacOS, Linux
```sh
pwd
```

Листинг текущей директории Windows: 
```sh
dir
```
Linux, MacOS:
```sh
ls
```
Удаление файла в Windows
```sh
del <filename>
```
в MacOS, Linux:
```sh
rm <filename>
```
